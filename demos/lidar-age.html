🌲 Tree age estimation from LiDAR

Drone LiDAR returns a dense 3-D point cloud; rasterise it and you get a canopy height model (CHM) — height above ground at every pixel. From structure alone we can detect individual tree crowns and, given a site-index curve, invert each crown's height into an estimated age. Adjust the detection threshold and site productivity and watch the stand-age estimate respond.

Inversion settings

CHM is a synthetic 64×64 raster (~1 ha). Crowns are real local-maxima detections on the surface; ages come from inverting a Chapman–Richards height–age model.

How this works — the method behind the demo

A CHM is the LiDAR digital surface model minus the bare-earth model. Individual trees appear as peaks; we detect crowns with a local-maximum filter — a pixel taller than all neighbours and above a height threshold (variable-window methods scale the window with height in production).

Stand age comes from inverting a site-index height–age curve. We model top height as Chapman–Richards H = Hmax·(1−e−k·age)p, anchored so H equals the site index at base age 50. Inverting for age:

age = −ln(1 − (H / Hmax)1/p) / k

Because LiDAR fixes the height, a more productive site (higher SI) implies a younger stand at the same height — slide the site index and the age estimate drops accordingly.
